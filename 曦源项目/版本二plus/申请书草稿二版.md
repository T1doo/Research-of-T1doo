## 四、课题开题报告

### 4.1 课题的目的和意义

视觉-语言-动作（Vision-Language-Action，VLA）模型作为机器人通用操作控制的新兴范式，已在仿真与真机任务中展现出较强的指令遵循与操作能力。然而，VLA 在标准基准上的优异成绩尚不足以反映其在真实部署环境中的可靠性：研究表明，现有模型对环境扰动（光照变化、背景纹理、相机视角、物体初始位置以及语言指令改写等）高度敏感，任务成功率往往由 90% 以上急剧下降至 30% 以下，这一鲁棒性瓶颈严重制约其从受控实验环境向真实应用场景的迁移。

针对上述问题，近年研究形成了两类受到广泛关注的技术路线：其一是在模型输出层引入显式空间提示，促使模型在生成动作前先对物体的空间关系作出判断；其二是在视觉特征层引入三维监督信号，使模型隐式习得空间感知能力。两类方法各有所长，但二者的协同效应，以及如何在多扰动评测与真机环境下对其增益进行系统验证，目前仍属研究空白。

本项目的研究意义在于：以一个开源 VLA 主干为统一基础，系统考察"显式方向词提示"与"区域加权三维表征对齐"两层互补机制在多扰动评测下的鲁棒性增益，并借助真机扰动评测补足从仿真到真实的迁移验证。研究结果可为后续 VLA 鲁棒性研究提供可复用的对照实验框架与开源工具，也为机器人操作系统由实验环境向实际应用场景迁移提供实证依据。

### 4.2 课题研究状况（国内外主要发展现状与趋势）

#### 4.2.1 视觉-语言-动作模型

视觉-语言-动作（VLA）模型将预训练的视觉-语言模型（Vision-Language Model，VLM）与动作解码器相结合，利用 VLM 蕴含的语义知识与视觉理解能力，实现从自然语言指令到机器人底层动作的端到端映射[1,2,3]。OpenVLA[1] 以 70 亿参数的 VLM 为主干，在离散化动作空间上自回归生成动作；RT-2[3] 验证了将网络规模的视觉-语言知识迁移至机器人控制的可行性。π₀[2] 提出了一个支持多种规模变体的灵活框架，其自回归变体 π₀-FAST 采用 PaliGemma[4] 作为视觉-语言主干（语言模型为 Gemma-2B），并引入 FAST[5] 动作 tokenizer，将连续动作离散化为语言模型可自回归生成的动作 token，以降低训练与推理成本；π₀-FAST 参数规模约 30 亿、开源且对 LoRA 等参数高效微调方法友好。本项目即以 π₀-FAST 作为统一主干（InSpire[6] 的真机实验亦以 π₀-FAST 为基线，二者链路天然契合；详见 §4.3）。

#### 4.2.2 显式空间提示机制

增强 VLA 空间感知能力的一条技术路线是在输出层引入显式空间推理环节，其假设是：若模型先显式判断物体间相对位置、再生成动作，可获得更可靠的空间先验，从而减少因视觉歧义或场景扰动导致的决策错误。InSpire[6] 是该路径的代表性工作：在原任务指令前自动插入一个方向性问题（如"目标物体相对参考物体的方位"），训练模型在生成动作 token 前先预测方向词，从而在相机视角变化、物体布局重排等扰动下提升鲁棒性。同类思路的外围工作还包括以人眼凝视作为空间注意力先验[7]、借助视觉语言模型自动生成视觉显著性图[8] 等。然而该类方法存在两点局限：方向词是离散、有限类别的空间抽象，能否覆盖真实场景中连续而复杂的空间关系尚待验证；且其监督仅作用于输出层，对模型内部视觉表征的形成缺乏直接引导。

#### 4.2.3 隐式三维表征对齐

与显式空间提示不同，另一条技术路线（本项目重点）不改变输出形式，而在表征层面注入三维几何先验：使 VLA 中间层的视觉表征与一个冻结的三维基础模型的输出对齐，从而在不改变输出形式的前提下隐式获得对物体三维结构、深度关系与空间布局的感知。Spatial Forcing[9] 是其代表：引入 VGGT[10] 作为冻结教师，将其输出的三维特征与学生模型在相同视觉 token 位置的隐状态做逐 token 余弦相似度对齐；该设计无需显式深度输入或点云数据，仅在训练阶段加入对齐损失，即可在多个仿真基准上提升数据效率与扰动鲁棒性。但 Spatial Forcing[9] 对所有视觉 token 施加均匀对齐监督，未利用任务相关性先验，难以缓解 VLA 的注意力涣散问题——这正是本项目区域加权对齐的改进出发点。

#### 4.2.4 三维基础模型

VGGT[10]（CVPR 2025）是一个可从任意数量输入图像（从单张到多张）直接推理相机参数、深度图、点图与点轨迹等稠密三维信息的统一框架；相较于传统的运动恢复结构（Structure from Motion，SfM）与多视图立体（Multi-View Stereo，MVS），它不依赖显式的多视图几何优化，而以大规模数据端到端学习从图像到三维的映射。围绕 VGGT[10] 已衍生出一系列加速与续作工作[11,12,13]。本项目选用其中的 FastVGGT[11] 作为冻结三维教师，依据是其与 VGGT[10] 原版接口完全兼容、可直接加载原 checkpoint 并平滑回退至原版（其约四倍加速主要见于长序列输入，本项目对 LIBERO[15] 单帧离线抽取教师特征几乎不享有该加速收益，单帧吞吐以预实验实测为准）。

#### 4.2.5 视觉注意力引导机制

自回归 VLA 在视觉信息利用上存在三个相互关联的瓶颈：架构偏差使模型倾向忽视视觉细节、过多的视觉 token 使注意力难以聚焦正确区域、任务无关的视觉信息引入大量噪声[14]。针对于此，FocusVLA[14] 提出将注意力导向任务相关视觉区域的范式，包含两项核心设计：模态级联注意力（消除语言与动作历史形成的捷径通路，迫使模型在动作生成时依赖任务相关视觉细节）与聚焦注意力（动态选择任务相关视觉 patch 以控制信息量，并显式调制其权重以抑制任务无关噪声）。实验表明，FocusVLA[14] 在仿真与真机基准上均能更有效地利用视觉细节执行灵巧操作并加速收敛。其"聚焦任务相关区域"的思想与本项目对三维对齐监督施加区域加权的动机一致，区别在于本项目作用于中间层表征对齐的监督权重而非注意力本身。

#### 4.2.6 鲁棒性评测基准

LIBERO[15] 提供了一个包含多类操作任务的仿真基准，但其各任务场景的环境参数（光照、相机视角、物体位置等）为固定设置，难以反映真实部署中的扰动条件。针对此不足，LIBERO-Plus[16] 在 LIBERO[15] 基础上构建了七维度扰动评测集（物体布局、相机视角、初始状态、语言指令、光照、背景纹理与传感器噪声），每个扰动维度独立可控，以单一变量方式考察模型的鲁棒性边界；该基准已被 InSpire[6]、Spatial Forcing[9] 等工作采纳为仿真评测标准，是目前最为系统的 VLA 鲁棒性仿真评测平台之一。

综上，现有工作分别覆盖了显式空间提示、隐式三维对齐、模型视觉利用与鲁棒性评测四个方面，但"显式方向词提示"与"隐式三维表征对齐"二者的协同效应，以及在多扰动评测与真机评测两轴下的系统验证，目前仍缺乏直接的实证研究。本项目旨在填补这一空白。

### 4.3 主要内容、基本思路、难点和创新点

#### （一）主要内容

本项目以开源自回归 VLA 主干 π₀-FAST（基于 π₀[2]，详见 §4.2.1）为统一基础，系统考察"显式方向词提示"与"区域加权隐式三维表征对齐"两层互补机制对模型鲁棒性的协同增益，研究内容分为四部分：

1. **显式方向词提示插件的实现**：参照 InSpire[6] 的设计思路，在 π₀-FAST 框架中完成任务指令前置方向性问题的提示改造、方向词分类头构建、方向词自动标注脚本开发与联合损失函数的工程化部署。
2. **区域加权三维表征对齐模块的设计**：以 FastVGGT[11] 作为冻结三维教师，对语言模型主干（Gemma-2B）中间层抽取的视觉 token 隐状态引入逐 token 余弦相似度对齐损失；在 Spatial Forcing[9] 基础上扩展区域加权掩码机制，实现对不同视觉 token 的差异化监督。
3. **双源融合的区域加权掩码构建**：融合方向词标签反向投影（任务相关性先验）与基于 DINOv2[17] 的视觉显著性（视觉显著性先验）两路信息，对任务相关区域施加更强、对背景区域施加更弱的对齐监督，提升监督信号的任务聚焦性。
4. **系统评测与机制分析**：在 LIBERO-Plus[16] 多扰动评测集与小规模真机扰动评测两个维度上，开展基线、仅显式提示、显式提示+区域加权对齐三变体的对照实验，并通过视觉利用消融实验探究各机制的作用机理。

#### （二）基本思路

在统一主干、统一训练数据与统一评测协议下，对三个模型变体进行对照实验。仿真评测以 LIBERO-Plus[16] 七维度扰动集为主要标尺（主对照聚焦相机视角、机器人初始状态等核心维度，其余维度作拓展评测），辅以原版 LIBERO[15] 作为干净基线；真机评测参照 InSpire[6] 的协议设计三个任务与三个扰动维度，累计完成三百余个真机评测序列。通过控制变量法量化两层机制各自的边际增益及其协同（叠加）效应，从而回答二者究竟是互补、冗余还是相互干扰这一开放经验问题。

#### （三）主要难点

1. **跨层联合优化的稳定性**：三维对齐损失须注入 Gemma-2B 中间层，与方向词、动作的自回归交叉熵目标联合优化。连续的对齐损失将中间层表征拉向三维几何空间，而同一批表征又必须服务于下游 next-token 预测，二者存在内在张力，需在损失权重、注入层位与训练策略上审慎设计并经预实验平衡。
2. **聚焦监督的有效性边界**：双源区域加权相较均匀对齐能否带来稳定增益尚无定论（同类注意力引导工作的效应量通常较小），需经来源消融界定最优权重配置与适用范围。
3. **自动生成监督信号的质量**：方向词标签由仿真轨迹与真机位姿自动推导，三维教师特征取自合成渲染域，二者均存在标注噪声与域偏移，构成方法的上限约束，须经小规模人工抽样核验与预实验把关。
4. **仿真到真机的迁移与小样本约束**：真机评测受人工标注预算限制，需在有限数据下完成可靠微调与鲁棒性检验；仿真侧以样本规模匹配的阈值与多重比较校正保证统计效力，真机侧则定位为初步真实世界证据（preliminary real-world evidence），以扰动维度上的方向一致性为主要判据。

#### （四）主要创新点

1. **协同机制的系统验证**：在统一自回归开源主干（π₀-FAST）与统一评测协议下，系统考察"显式方向词提示"与"区域加权隐式三维表征对齐"的协同（叠加）效应。二者分走输出层离散方向词监督与中间层连续表征对齐两条不同通路，其互补、冗余抑或相互干扰的关系无法由各自单独结果推断；据现有检索，尚无在同一框架内的协同验证工作，本项目填补这一空白。
2. **区域加权对齐监督机制**：针对 Spatial Forcing[9] 对全部视觉 token 施加均匀对齐、未利用任务相关性先验的局限，提出由方向词标签反向投影先验与基于 DINOv2[17] 的视觉显著性先验双源融合、按前景/上下文/背景分层加权的区域加权掩码，对 per-token 对齐损失权重做"任务相关性 × 视觉显著性"双源调制——区别于同族并行工作 GLaD[18] 的特征级加权融合（本项目作用于对齐损失权重而非特征），并经来源消融论证其优于均匀监督。
3. **可复用的统一对照框架**：在 LIBERO-Plus[16] 多扰动与真机扰动两轴上，以可证伪假设与统计检验系统量化两层机制的鲁棒性增益，形成三变体统一对照的可复用实验框架与开源工具，为后续 VLA 鲁棒性研究提供基础平台与基线参考。

### 4.5 课题预期成果

1. 一份扰动鲁棒性实证研究报告，整理为 4-6 页国际研讨会短稿，目标投稿至 CoRL 2027 / ICLR 2027 Robot Learning workshop 等机器人学习方向的研讨会。
