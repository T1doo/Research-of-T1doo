# 版本三 · 立项调研索引

> **项目定位**：**基于物体分割引导的视觉-语言-动作模型视觉聚焦质量与扰动鲁棒性诊断研究**
> **内部代号**：**SAM-as-Probe**
> **状态**：立项调研期（2026-05-26 起）
> **团队**：2 名大二本科生 · ¥10K 预算 · 1 年（2026-06 → 2027-06）

## 一句话项目定位

> 以 SAM2/GroundedSAM 物体 mask 作为「外部视觉真值」，对主流 VLA（OpenVLA-OFT / SmolVLA）在 LIBERO clean + LIBERO-Plus 7 维扰动下做 "视觉聚焦质量 × 扰动鲁棒性" 的**纯诊断式实证研究**——不训练 VLA，把 SAM 当探针。

## 与版本一二的关系

- **版本一**：Efficient VLA（AAC + RobustVLA + RC-NF 三方向耦合）→ 因 4-6 人假设、工程过载放弃
- **版本二**：FocusVLA × InSpire 互补（显式提示插件 LoRA 微调 + 视觉利用 ablation）→ 工作版 v2 已完成
- **版本三**（当前）：**全新路线**——把 SAM 当探针，零训练，对 VLA 视觉聚焦质量做外部真值评测

详见 [[09_与版本二差异化说明]]。

## 核心叙事一句话

> "FocusVLA 等近期工作明确把 VLA 鲁棒性差归因于'视觉利用机制'，但这些工作自身都没在 LIBERO-Plus 等扰动评测平台上**用外部视觉真值**系统验证"模型真的看对了任务相关区域"。本项目以 SAM2/GroundedSAM 物体 mask 为外部参考真值，对主流 VLA 做'视觉聚焦质量 × 扰动鲁棒性'的纯诊断式实证研究，回答一个被绕过的根问题：**VLA 的视觉聚焦质量是否真的与扰动鲁棒性有因果关系**。"

## 关键决定一览

| 维度 | 决定 |
|---|---|
| 主推方案 | **SAM-as-Probe 纯诊断式实证研究**（零训练、attention hook + SAM mask + IoU） |
| 备选方案 | [[02_备选方案_仿真GT_mask兜底]]（GroundedSAM 失效时用 LIBERO 仿真 GT mask） |
| 排除方案 | 复现 Oat-VLA / 真改 FocusVLA 架构 / SAM 弱监督训练 VLA（全因需重训，与"纯评测"约束冲突） |
| Backbone | **OpenVLA-OFT-7B（主） + SmolVLA-0.5B（备，速度 5×）** |
| Benchmark | LIBERO-Plus 7 维（主战场） + LIBERO clean（baseline） + LIBERO-Para / VLA-Risk（stretch） |
| 核心评测对象 | 2 backbone × 8 条件（clean + 7 维） × 500 instance × 3 seed |
| 外部视觉真值来源 | 轨道 A：GroundedSAM + SAM2 时序传播；轨道 B：robosuite 仿真 GT mask（兜底） |
| 投稿目标 | CoRL/ICLR/NeurIPS Robot Learning workshop（4-6 页短稿） |

## 三个核心研究假设速览

- **H1（对齐假设）**：失败 episode 的 attention × SAM mask IoU 显著低于成功 episode，ΔIoU ≥ 0.10
- **H2（理论最优 K）**：覆盖任务物体 mask 所需的最小 K\* 中位数 ≤ 80，固定 K=256 在多场景过度选取
- **H3（联动崩塌）**：SAM mIoU drop 与 VLA SR drop 正相关 r ≥ 0.5；同步失败 episode ≥ 30%

详见 [[01_主方案_SAM探针式诊断]] § 2 与 [[08_最终方案_SAM_as_Probe]] § 3。
