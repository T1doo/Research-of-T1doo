# 版本三 · 可行性与 Novelty 分析报告

> **本文档独立于立项调研笔记 + 申请书草稿**，目的是用一份可独立阅读的材料，**说服老师与团队成员相信**版本三（SAM-as-Probe 纯诊断式实证研究）这条路线确实存在文献空白、确实在 2 大二本科生 + ¥10K 预算 + 1 年期内可执行、且与版本二（InSpire 显式提示插件 + 视觉利用 ablation）的差异化清晰。

## A. User Idea 的两条原始思路梳理

### A.1 思路一：把 visual token 做成变长

**原始动机**：FocusVLA 等"视觉聚焦类 VLA"工作中 visual token 数量是**固定 K=256**（消融实验认为 K=128 不够、K=512 多余）。直觉上，不同场景应当需要不同的 K——简单场景少看一些、复杂场景多看一些。

**现有工作**（按"K 是否真正变长"分类）：

| 类别 | 代表工作 | K 处理方式 |
|---|---|---|
| **真正变长 K** | TokenFLEX (arXiv:2504.03154) | 训练时随机 K ∈ {64, 144, 256}，推理时可灵活选 |
|  | One Token Per Frame (arXiv:2605.07931) | 极端压缩：每帧仅 1 个 token，自适应注意池化 |
|  | DyVTE (arXiv:2411.19628) | layer-wise early exit，某层判断视觉 token 是否还有用 |
|  | VLA-Cache (arXiv:2502.02175, NeurIPS 2025) | 隐式变长：静态 token 缓存复用，计算量变了 |
| **物体数自适应（接近变长）** | Oat-VLA (arXiv:2509.23655) | object-agent pooling，256→16 token（但 16 是固定值）|
|  | SlotVLA (arXiv:2511.06754, ICRA 2026) | slot attention 学物体表示 |
| **固定 K + 动态 schedule** | VLA-IAP (arXiv:2603.22991) | training-free，按任务阶段调剪枝强度（仍固定 K）|
|  | VLA-Pruner (arXiv:2511.16449) | dual-level scoring + top-K |
|  | Compressor-VLA (arXiv:2511.18950) | instruction-conditioned compression |
|  | SemanticVLA (arXiv:2511.10518, AAAI 2026 Oral) | SD-Pruner 双 encoder |
|  | LUVC (arXiv:2512.09010) | spectral pruning，lossless 2× |
|  | TPRL (arXiv:2603.13394) | RL 学固定策略 |

**结论**：变长 K 这一条具体技术路线已被 TokenFLEX、VLA-Cache、DyVTE 等工作覆盖；物体数自适应已被 Oat-VLA、SlotVLA 覆盖。**作为"新方法"做工程开创没有空间**，必须从另一角度切入。

### A.2 思路二：接入物体分割模型（SAM2 / GroundedSAM）

**原始动机**：让模型显式知道"任务物体在哪"，避免 attention 走捷径。

**现有工作**：

| 工作 | 怎么用 SAM | 与本项目的关系 |
|---|---|---|
| **Grounded SAM** (arXiv:2401.14159, IDEA-Research) | GroundingDINO + SAM 标准管线（text → box → mask） | 工程级工具，本项目轨道 A 直接复用 |
| **SAM2Act** (arXiv:2501.18564) | SAM2 做 spatial memory | 用了 SAM2 但非 token 替代，与本项目"探针"姿态不同 |
| **AxisEst** (arXiv:2409.16287) | SAM2 视频追踪 articulated 物体 | SAM2 在仿真上的工程验证 |
| **ReKep / VoxPoser** | SAM 做 keypoint 区域提取 | SAM 是上游工具，不是 VLA 内部模块 |
| **ST-VLA** (arXiv:2603.13788) | 4D 时空 mask 作为 VLA condition | mask 在策略里用，需要重训 ST-VLM |
| **GuidedVLA** (arXiv:2605.12369, RSS 2026) | 提到用 SAM/GroundingDINO 自动标注 object bbox 作 supervisory signal | 思路相邻，但需要训练 |

**结论**：把 SAM mask 接入 VLA 已经有多种方式（spatial memory / keypoint / 4D condition / 弱监督），但都需要**改架构或重训 VLA**。**作为"新方法"也已经被多角度覆盖**。

### A.3 两条思路的共同问题

**用户的原始两条思路都已经被部分做过，但都集中在 efficiency / capability 的提升维度，没有一篇论文真正回答："这些视觉聚焦机制的鲁棒性如何？"**

这就是本项目的切入角度——从"做新方法"转向"做新评测/诊断"。

## B. 文献留白：5 个未被填补的研究问题

通过整合 `论文探索/` 101 篇笔记 + 调研期 web 检索的 15+ 篇相邻工作，明确识别出 5 个文献空白：

| # | 空白 | 影响范围 |
|---|---|---|
| 1 | **没有论文用 SAM/GroundedSAM mask 作为外部参考真值**系统评估 VLA 视觉聚焦的**正确性**与**鲁棒性** | 全行业 |
| 2 | **FocusVLA、Oat-VLA、SlotVLA 等"视觉聚焦类 VLA"完全没在 LIBERO-Plus 等扰动 benchmark 上做 attention-mask IoU 量化** | FocusVLA 自述局限 #3 #4 |
| 3 | **没有论文研究"扰动下 VLA 注意力 vs 物体真实 mask 的 IoU 退化"** | 鲁棒性方向所有工作 |
| 4 | **没有论文用 SAM mask 计算"理论最优 K\*"** 评估固定 K=256 是否合理 | 所有 token pruning 工作 |
| 5 | **没有论文做"SAM 也认错（共同失败）vs SAM 认对但 VLA 仍崩（看错了）"的失败模式诊断** | failure detection 方向 |

### 项目内的预先工作证据

本研究方向并非临时拍脑袋——`论文探索/A_视觉利用鲁棒性/papers/2024-01_GroundedSAM_Ren.md` 末尾的 **Action Item** 早在 2026-05 调研初期就明确记录：

> 「**评估：FocusVLA attention heatmap vs Grounded SAM mask 的 IoU 在 LIBERO 上量化**」

版本三正是把这条 Action Item 系统化、扩展到 LIBERO-Plus 7 维扰动并加上 H2 K\* 与 H3 联动崩塌两个衍生维度。

### 关键 novelty 论证

本项目**没有任何一项技术原创**——attention hook、SAM2、IoU 都是现成工具。**但没有任何一篇论文同时**：
1. 用 SAM mask 作外部真值评估 VLA 视觉聚焦
2. 在 LIBERO-Plus 7 维扰动下系统化
3. 同时报告 IoU + K\* + 联动崩塌三个互补维度

这是个**"组合空白"**，对 2 大二本科生而言：(i) 没有技术原创压力 (ii) 工程量友好（拼现成工具）(iii) 故事完整（三个互相支撑的发现）。

## D. 可行性评估（4 维度）

### D.1 GPU 算力（核心约束）

**预算估算**：
- LIBERO-Plus 7 维 × 2 backbone × 500 instance × 3 seed ≈ 24K episode
- 每 episode ≈ 30s（SmolVLA 主跑，OpenVLA-OFT 关键 cell 校验）
- 加上 SAM2 推理（每 episode 取 5 keyframe + 视频传播）≈ 5s
- 总计 ≈ 200 GPU-h 单 A100

**¥10K 预算覆盖**：
- 云端 A100 约 ¥25-35/h（AutoDL / 矩池云学生价）
- 200h × ¥30 = ¥6,000，**在预算内**
- 留 ¥4,000 给投稿 + API + 数据存储等

**风险**：评测规模可能 underestimate。**分层抽样**（次要 3 维取 1/3）可压到 ~120 GPU-h，进一步留出余量。

**结论**：✅ GPU 预算友好。

### D.2 工程难度

**核心技术栈**：
- ✅ PyTorch attention hook：已有公开教程，约 50 行代码
- ✅ HuggingFace `transformers` 加载 VLA：标准操作
- ✅ Grounded SAM 官方 demo：15K+ GitHub stars，开箱即用
- ✅ SAM2 官方 video predictor：facebookresearch/sam2 文档完整
- ✅ IoU 计算：4 行 numpy 代码
- ✅ LightGBM 二分类器：标准 ML pipeline，100 行

**主要工程坑**（已识别，可缓解）：
1. **OpenVLA-OFT 内部 attention 层选择**——last-layer cross-attention 不一定是 attention 质量最好的层。**缓解**：W1 末做层选择 ablation（last layer vs penultimate vs 多层平均）
2. **SAM mask → patch grid 投影的阈值**——0.3 / 0.5 像素覆盖阈值如何选。**缓解**：W1 末做 ablation
3. **GroundedSAM 在 LIBERO 渲染图上的精度**——LIBERO 不是 COCO 真实图像分布。**缓解**：P1 末校准实验 + Plan B 兜底

**结论**：✅ 难度对 2 大二本科生**完全可控**，与版本二（需要写方向词 GT 标注脚本 + cls head + loss 联合训练）相比工程量降低 ~50%。

### D.3 数据来源

| 数据 | 来源 | 状态 |
|---|---|---|
| VLA backbone | HuggingFace（openvla/openvla-7b-oft + HuggingFaceTB/SmolVLA-base） | ✅ 已开源 |
| LIBERO 原版 | github.com/Lifelong-Robot-Learning/LIBERO | ✅ 已开源 |
| LIBERO-Plus 7 维 | HuggingFace `senyufei/LIBERO-Plus` | ✅ 已开源 |
| SAM2 权重 | facebookresearch/sam2 | ✅ 已开源 |
| GroundingDINO + SAM 权重 | IDEA-Research/Grounded-Segment-Anything | ✅ 已开源 |
| LIBERO+ mask 标注（备用） | SlotVLA 论文（ICRA 2026） | ⏳ 待确认 release |

**结论**：✅ 全部数据资源已开源，无需自己采集/标注。

### D.4 风险全谱

| 风险 | 严重度 | 概率 | 缓解 |
|---|---|---|---|
| GroundedSAM 在 LIBERO 渲染图上失效（mIoU < 0.5） | 🔴 高 | 40% | P1 末校准实验 → 切轨道 B（仿真 GT mask） |
| H1 完全否定（IoU 与 SR 无关） | 🟡 中 | 25% | 这是反直觉发现，可投 negative-finding workshop |
| 评测预算超支（350 GPU-h > 200h） | 🟡 中 | 35% | 分层抽样 + SmolVLA 主跑 + OpenVLA-OFT 子集复现 |
| Attention hook 与现有 backbone 不兼容 | 🟢 低 | 15% | OpenVLA-OFT 已开源，hook 是标准操作 |
| H2 完全否定（K\* 中位数 > 200） | 🟢 低 | 15% | 仍是有价值发现"固定 K 合理" |
| FocusVLA 代码至 P4 末仍未 release | 🟢 低 | 50% | 不依赖 FocusVLA 复现，stretch 加第 3 backbone |
| 团队工程能力跟不上 | 🟢 低 | 20% | 工程量比版本二低，且大部分是"拼接现成工具" |

**结论**：✅ Top 1 风险（GroundedSAM 失效）有明确兜底方案；其他风险都可缓解。

## E. 三种 Scope 备选（与导师可讨论）

为给导师 / 团队留选择空间，提供 3 种 scope 备选：

### E.1 Conservative Scope（最稳，对应 Plan B 本身）

> 直接走 Plan B（LIBERO 仿真 GT mask），不依赖 SAM 工程。研究问题：**VLA 视觉聚焦质量 × 扰动鲁棒性**（H1 + H2 + H3'）。投 workshop 短稿。

- **优点**：mIoU = 1.0 上限，H1 / H2 验证更严格；工程量更小；零外部依赖
- **缺点**：真机泛化承诺减弱；评审可能觉得"用仿真 GT 不算新方法"

### E.2 Recommended Scope（中等，对应 Plan A 主推）

> Plan A：GroundedSAM + SAM2 主路线，P1 末校准实验决定是否切轨道 B。研究问题：H1 + H2 + H3。投 workshop 短稿。

- **优点**：novelty 更强（SAM 是通用工具，论证可推广到真机）；故事完整
- **缺点**：依赖 GroundedSAM 精度，存在切换风险

### E.3 Ambitious Scope（最激进，stretch）

> Plan A + 加第 3 backbone（FocusVLA，若 release）+ 真机验证（小规模，借实验室真机数据）+ 加入 VLA-Risk 6 维评测。投 CoRL main 短稿。

- **优点**：故事最完整，投 main 短稿可能
- **缺点**：依赖 FocusVLA 代码 release + 真机访问；超 2 本科生承受能力，**不推荐**

### 推荐选 E.2

理由：(i) novelty 与可行性平衡最佳；(ii) Plan A → Plan B 切换成本低，相当于自带保险；(iii) 与导师"FocusVLA 方向深挖鲁棒性"的指引最对齐。

## F. 一句话结论

> **版本三（SAM-as-Probe 纯诊断式实证研究）填补的是一个"组合空白"——用 SAM mask 当外部视觉真值、在 LIBERO-Plus 7 维扰动下、同时报告 IoU/K\*/联动崩塌三个互补维度——这个组合在 2024-2026 文献中没有任何一篇论文同时做过。工程量比版本二低约 50%（零训练、零方向词 GT、零 cls head），与导师"在 FocusVLA 方向深挖鲁棒性"的指引完全对齐（直接回应 FocusVLA 自述局限 #3 #4）。对 2 大二本科生 + ¥10K 预算 + 1 年期，Plan A → Plan B 双轨保险下，可执行性接近版本二 Plan B 兜底路线但 novelty 与故事完整度更强。建议采纳。**
