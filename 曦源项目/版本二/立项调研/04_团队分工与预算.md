### GPU 来源策略（按优先级）

1. **学校共享 GPU**（首选）— 复旦大学 / 学院计算资源（如有），免费
2. **AutoDL / 矩池云**（国内云）— A100 80GB 约 ¥6-8/h，性价比高
3. **腾讯云 / 阿里云**（备用）— 价格略高但稳定
4. **Lambda Labs / RunPod**（境外，需要外汇）— 仅作万不得已

### 投稿费用估算

## 5. 风险与缓解（与预算相关）

| 风险 | 缓解 |
|---|---|
| GPU 价格上涨（最近 RTX 4090 / A100 都涨） | 优先用学校资源；提前 1 个月预订云端实例 |
| 训练翻车需要 redo（最大成本风险） | 备用金 ¥1,300 = 40 GPU-h，足够 1-2 次完整 redo |
| API 使用失控（写代码时 Claude 嗨了） | 每周末统计 API 用量，超 ¥150/周预警 |
| 投稿不中需要多次重投 | 国内会议作为最后保底（注册费可控） |
