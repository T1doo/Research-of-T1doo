## P2+P3 完成总结（2026-05-26 第一阶段）
- **总笔记数：95 篇精读笔记**（A:28, B:32, C:11, Bench:11, Classic:13）+ 锚点深度笔记 2 篇（FocusVLA / VLA-Risk）

- **核心叙事/创新点**（agent 跨主线发现）：
  1. 「诊断（A6 ShortcutLearning）—架构（A FocusVLA）—数据（B RobustVLA）」三方组合 — BC 申请书叙事支点
  2. **「扩展 RobustVLA 到 joint cross-modal worst-case」**反 VLA-Fool 的独立扰动假设盲点
  3. **「VLM Verifier + RC-NF safety filter + TT-VLA」**（首推方案 3 核心）

  5. **DRO Library 一行代码替换** RobustVLA 多臂 bandit
  6. **InSpire** 是本科生 4-6 周 warmup 实验 ROI 最高（plugin 形式零数据）
  7. **RoboCasa + MimicGen 4-8 周可产出**闭环

### Agent A1+A2+A3 完成洞察
**子主题趋势**：
- **A1 Token Pruning**：演化为「dual-objective / task-phase aware / RL-learned」三条路线；**共同空白：均未在视觉扰动 / OOD benchmark 上系统评测剪枝对鲁棒性的影响** → FocusVLA 主线可差异化切入
- **A2 Attention Guidance**：oracle 信号源迭代越来越便宜（人眼 gaze → VLM saliency → DINOv2 self-supervised）；AutoFocus-IL "VLM > 人类 gaze" 是 striking claim
- **A3 Patch Selection**：从 2D attention → 3D（SpatialVLA Ego3D PE）→ 4D（ST-VLA 44.6% / 30.3% zero-shot 提升）

**最值得后续深挖**：
1. **TraceVLA**（2024-12_TraceVLA_Zheng）— 4B 打败 7B，与 FocusVLA 正交可叠加；3.5× 真机提升
2. **AutoFocus-IL**（2025-11_AutoFocus-IL_Gong）— 零成本 attention 监督，可作 FocusVLA plug-in

**待补**：EF-VLA 论文 ID 未能定位（OpenReview / arXiv 都没找到匹配），已写占位笔记。

### 主线A 搜寻心得
**热点**：A1 token pruning 与 A3 patch selection 最热，2024-2026 新作扎堆；DINOv2+SigLIP 已成标准；A6 spurious correlation 从 2024 中开始成为高频话题。
**空白**：CF-VLA 谱系刚起步；Perceiver IO 在 robot 上的探索有限；grounding+VLA 端到端联合优化未成体系。
**与 FocusVLA 相关性最高**：A1 (VLA-Pruner, LUVC) + A3 (EF-VLA, GuidedVLA, SpatialVLA)，这些是 FocusVLA 的"同代竞争者 / 跟随者"。
